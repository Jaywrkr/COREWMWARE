¿Qué es VMware Cloud Foundation?
Una nube privada completa.
En tu datacenter. Bajo tu control.
VCF no es solo virtualización. Es el stack completo — cómputo, red, almacenamiento, seguridad, Kubernetes y automatización — integrado, certificado y operado como una sola plataforma. Lo mismo que AWS o Azure, pero en tu propiedad.
1
Cómputo — vSphere 9
ESXi · vCenter · vMotion · HA · DRS
▼
El motor de virtualización más maduro del mercado. vSphere HA reinicia VMs automáticamente si un host falla — sin que el usuario note nada. vMotion migra VMs en caliente entre servidores en segundos. DRS balancea la carga automáticamente. Soporta 92 VMs por host físico vs 52 de Hyper-V — mismo hardware, más densidad. [2]
2
Almacenamiento — vSAN ESA
All-NVMe · SPBM · compresión nativa · sin SAN externa
▼
Convierte los discos NVMe de tus propios servidores en un pool de almacenamiento distribuido. Elimina la SAN externa y su costo. vSAN ESA (Express Storage Architecture) soporta deduplicación, compresión y cifrado nativo. Políticas de almacenamiento por VM (SPBM): defines rendimiento y redundancia por aplicación, no por disco. [7]
3
Red y Seguridad — NSX Enterprise
SDN · L2/L3 · load balancer · firewall distribuido* · micro-segmentación*
▼
Red y seguridad definidas por software. NSX crea redes virtuales sin tocar switches físicos — VLANs, routing L2/L3, load balancer y VPN incluidos. Nota: El firewall distribuido (NSX DFW) y micro-segmentación son funciones del add-on de seguridad avanzada de NSX — no están incluidos en VCF base.* [7]
4
Kubernetes y Automatización
VKS Enterprise · Harbor Registry · VCF Automation · IaC
▼
VKS Enterprise (vSphere Kubernetes Service) provee clústeres Kubernetes nativos sobre el mismo hipervisor — sin agentes externos ni plataformas separadas. Harbor Registry incluido para imágenes de contenedores. VCF Automation (antes vRA) permite aprovisionar VMs, redes y almacenamiento desde código (Terraform/Ansible) o un portal de autoservicio para los equipos de desarrollo. [7]
5
Gestión unificada — SDDC Manager
lifecycle management · parches orquestados · visibilidad total
▼
El cerebro operativo de VCF. SDDC Manager orquesta el ciclo de vida de todo el stack: valida compatibilidades, descarga parches, aplica actualizaciones en orden seguro y hace rollback si algo falla — todo sin intervención manual. Incluye también VCF Operations (observabilidad, alertas, capacity planning) y acceso a Broadcom Academy (cursos bajo demanda para todo el equipo, sin costo adicional). [5]

Tecnologías clave de eficiencia en vSphere

DRS — Distributed Resource Scheduler

Balancea VMs entre hosts del clúster automáticamente según carga real, migrando workloads sin interrupción vía vMotion. [5]

NUMA-Aware Scheduling

Coloca VMs donde la RAM es más accesible para el CPU, reduciendo latencia en servidores multi-socket enterprise. [5]

Memory Overcommit (TPS + Ballooning)

Permite asignar más RAM virtual que física usando deduplicación de páginas idénticas entre VMs. Ratio 3:1 sostenible en producción. [1]

vMotion en caliente

Migración de VMs activas entre hosts sin downtime, permitiendo mantenimiento y actualizaciones sin ventanas de impacto. [6]

Memory Tiering (vSphere 8+)

Extiende la memoria efectiva usando NVMe o PMem como nivel secundario de RAM. Reduce hasta un 40% la DRAM necesaria — impacto directo en CAPEX cuando la RAM está por las nubes. [1]

Planes VCF · VVF vs VCF
¿Cuál es el plan
que te corresponde?
Para empresas medianas y grandes, la decisión relevante es entre VVF (virtualización enterprise) y VCF (nube privada completa). La diferencia no es solo de características — es de modelo operativo. [7]
VVF
vSphere Foundation — Virtualización enterprise
  • vSphere 9 (ESXi + vCenter)
  • vMotion / HA / DRS
  • vSphere Distributed Switch (vDS)
  • vSphere Lifecycle Manager (vLCM)
  • vSphere Kubernetes Service (VKS) básico
  • vSAN OSA (All-Flash opcional)
  • Trust Authority
  • Site Recovery Manager (SRM)
Nube Privada Completa
VCF
VMware Cloud Foundation — Stack enterprise completo
  • Todo lo de VVF incluido
  • NSX Enterprise + Gateway Firewall
  • vSAN ESA — sin límite de hosts
  • VKS Enterprise + Harbor Registry
  • VCF Automation — ilimitado
  • VCF Operations + Log Intelligence
  • APM (Application Performance Monitoring)
  • Private AI Foundation (cargas GPU)
  • SDDC Manager — ciclo de vida unificado
  • HCX — migración hacia/desde cloud pública
